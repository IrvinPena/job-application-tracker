# Job Application Tracker

A backend web application built with **FastAPI** to help users track job applications, statuses, and notes.

This project is designed to demonstrate backend development skills including API design, database integration, and version control using Git and GitHub.

---

## Tech Stack

- **Python**
- **FastAPI**
- **Uvicorn**
- **SQLite**
- **SQLAlchemy**
- **Git & GitHub**
